PumpBros Whitepaper

(Draft v1.0)

Abstract

PumpBros is a browser-based 2D fighting game that combines competitive gameplay, wallet-gated access, online PvP battles, and community-driven rewards. Built on the Solana ecosystem, PumpBros allows eligible token holders to access the game, compete against AI or other players, unlock fighters through an in-game store, and participate in a transparent reward economy.

The project focuses on creating a fun competitive game first, while using blockchain technology only where it adds meaningful value: identity, ownership, rewards, and wager verification.

Vision

Most blockchain games fail because they prioritize speculation over gameplay.

PumpBros follows a different approach:

Gameplay First
Blockchain Second

The objective is to build a genuinely enjoyable fighting game where blockchain infrastructure enhances the player experience rather than replacing it.

Players should want to play because the game is fun. Rewards, ownership, and competition are additional layers that increase engagement.

Core Principles

Gameplay First

The fighting game experience remains the primary focus.

Fair Competition

Skill should be rewarded more than spending power.

Transparent Rewards

Reward distributions and leaderboard calculations will be public and verifiable.

Accessible Ownership

Players can unlock characters without requiring NFTs.

Community Growth

Long-term rewards are designed to encourage active participation rather than passive holding.

Platform Overview

PumpBros is a browser-based game playable worldwide.

Supported devices:

  • Desktop browsers
  • Mobile browsers (future)
  • Web-enabled gaming devices (future)

No installation is required.

Players connect a supported Solana wallet and immediately gain access to the ecosystem.

Wallet Integration

Supported wallets:

  • Phantom
  • Solflare
  • Backpack

Wallets serve as:

  • Authentication
  • Identity verification
  • Character ownership verification
  • Reward distribution destination

The wallet address acts as the permanent player identifier.

Token Holder Access

To participate, players must hold a minimum amount of the project's designated SPL token.

Example:

Minimum Requirement:
100,000 Tokens

Wallet eligibility is verified through blockchain data before access is granted.

This creates alignment between the game's player base and its supporting community.

Player Profiles

Every player creates a profile after connecting a wallet.

Profile data includes:

  • Username
  • Wallet address
  • PvP statistics
  • PvE statistics
  • Character ownership
  • Seasonal rankings

The wallet remains the source of truth for player identity.

Player vs CPU

Features:

  • Score tracking

Players compete against other players in real-time matches.

Features:

Security

PumpBros uses multiple layers of verification.

Wallet Signature Authentication

Players must prove ownership of their wallet.

Token Gate Verification

Token balances are validated before access.

Transaction Verification

Purchases and wagers are verified on-chain.

Replay Protection

Transaction signatures cannot be reused.

Match Audit Records

Every wager match is permanently recorded.

Match Audit System

Every wager match creates an immutable record.

Stored information includes:

  • Players
  • Wager amount
  • Deposit transactions
  • Winner
  • Payout transaction

Benefits:

  • Transparency
  • Fraud prevention
  • Dispute resolution
  • Statistical analysis
